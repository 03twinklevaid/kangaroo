# kangaroo-app

## Commands

### `npm run serve`

Runs the app in the development mode.<br />
Open [http://localhost:8080](http://localhost:8080) to view it in the browser.

### `npm run test:unit`

Runs the unit tests.

### `npm run test:e2e`

Runs the e2e tests.

### `npm run lint`

Run eslint

### `npm run lint-fix`

Run eslint and fix issues

### `npm run build`

Builds the app for production.
